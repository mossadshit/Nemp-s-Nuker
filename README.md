# Nemp's Nuker!

How 2 Install Nemp's Nuker:

```
git clone https://github.com/mossadshit/Nemp-s-Nuker.git
nano config.json
pip install -r requirements.txt
```

And you need to configure:
```json
{
    "Prefix": "",
    "Token": "",
    "Message": "",
    "ServerName": "",
    "ChannelName": "",
    "RoleName": "",
    "Amount": ""
}
```

Run:
```
python3 main.py
```
